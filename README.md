##  1、fs-table是什么

> 一种中后台业务表格解决方案，基于ElementUI表单、表格、分页组件二次封装table组件全量业务场景

##  2、fs-table有什么优势

> 10分钟可快速搭建crud页面

##  3、fs-table是否适合你的项目

> 如果你的项目是中后台且有大量使用表格页面场景（比如RouYi-Vue项目），可以考虑选择使用fs-table，只需引入并根据需求配置一份数据结构即可

## 4、如何查看Demo

``` bash
npm install / yarn

npm run dev / yarn dev

npm run build / yarn build
```
